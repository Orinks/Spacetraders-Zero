# SpaceTraders Zero Knowledge

## Project Overview
Zero-player game automating SpaceTraders.io API interactions. Focus on accessibility and screen reader support.

## Key Goals
- Provide accessible interface for SpaceTraders
- Automate agent activities
- Support screen readers
- Real-time status updates

## Technical Details
- Python with wxPython UI
- Environment variables for configuration
- SpaceTraders API v2 integration
- Windows-based development environment
  - Use Windows commands (e.g. `venv\Scripts\activate`)
  - Use backslashes for paths

## Development Guidelines
- Maintain screen reader accessibility
- Keep UI responsive
- Handle API errors gracefully
- Log important state changes
